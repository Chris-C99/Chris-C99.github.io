My Projects
File Converter
This was my final project for Brookdale Community College. It runs in the shell and can convert text to images and vice versa as a form of encryption. It was written in C++ and uses MYSQL to store certain information about the converted file such as the file size to the local database to comply with the project requirments.
Ripple Price Companion
This was an independent project I made to gain experience using android studio and creating android apps. The project used the retrofit interface to pull the price of a cryptocurrency called ripple from coindesk's API, and display it. The app also showed relevant data such as the daily, weekly and monthly gain percentages. I used java to create this app.
Train Reservation Web App
This project was made during a Data Managment class at Rutgers University. This project allowed customers to reserve train tickets and monitor train schedules. The database behind the project used MYSQL to store customer information and track the status of reservations. The front end of the website used JSP and CSS.
